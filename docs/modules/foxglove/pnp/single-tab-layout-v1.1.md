# PnP 单选项卡布局草案 V1.1

## 布局目标
在单一选项卡内完成“可打性判断、几何排障、诊断追因”，减少切换成本。

## 分区
- 顶栏主看板：22%
- 中层几何区：56%
- 底层诊断区：22%

## 分区详情

### 顶栏主看板（默认展开）
六项核心指标卡片：
- yaw
- pitch
- distance
- reproj_error
- is_solved
- meta

### 中层几何区（默认展开）
左右分栏 7:5：
- 左：重投影叠加图（原始角点 + 重投影点 + 误差线）
- 右：3D 姿态与短历史轨迹

### 底层诊断区（默认折叠）
- 延迟曲线
- 失败原因统计
- 活跃目标数
- 配置快照

## 刷新策略
- 数值层：全帧率。
- 几何层：15-20Hz，异常短时升频。
- 诊断层：低频 + 事件触发。
